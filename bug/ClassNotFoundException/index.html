# java.lang.ClassNotFoundException:javax.xml.bind.DatatypeConverter

报错描述：

java.lang.ClassNotFoundException:javax.xml.bind.DatatypeConverter

原因：可能是因为 SpringBoot 项目结合 JWT 进行登录时出现的问题，因为 jdk 版本太高导致的。

# 解决方案：

# 一，降低 jdk 版本
# 二，在 maven 中添加依赖

因为 javax.xml.bind 在 jdk8 中 有，但是在更高版本就没有了，所以我们加上就行了。
