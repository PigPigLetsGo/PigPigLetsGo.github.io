# 阻塞队列 - 单锁实现 - 防止交错执行 1

在 java 中要防止 代码 多线程下的 交错执行，我们就需要用到锁

在 java 中 锁 有两种选择：

  1. 一种锁的实现是 sychronized 关键字，功能较少
  2. ReentrantLock 翻译：entrant (进入) Reentrant (重新进入) Lock (锁) ReentrantLock (可重入锁) 。JDK 提供的工具类 而不是 关键字，相对功能比较多
