# 跳过测试

  • idea 中跳过测试

1. 选择测试 test 点击闪电图标

  • 使用 maven 指令跳过测试
mvn [指令] -D skipTests

或

mvn [指令] -Dmaven.test.skip=true

注意事项:

执行的项目构建指令必须包含测试生命周期，否则无效果，例如执行:compile 生命周期，不经过 test 生命周期

  • 细粒度控制跳过测试
