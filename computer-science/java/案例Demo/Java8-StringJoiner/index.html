# 介绍

StringJoiner 是 java8 新出的一个类，用于构造由分隔符的字符序列，并可选择性地从提供的前缀开始和以提供的后缀结尾。省的我们开发人员再次通过 StringBuffer 或者 StringBuilder 拼接
