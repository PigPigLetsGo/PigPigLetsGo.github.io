# 数据脱敏

# 1 功能

对应一些敏感数据进行脱敏

# 2 DesensitizedUtil 支持的脱敏数据类型

  • 用户 id
  • 中文姓名
  • 身份证号
  • 座机号
  • 手机号
  • 地址
  • 电子邮件
  • 密码
  • 中国大陆车牌，包含普通车辆，新能源车辆
  • 银行卡

# 3 使用频率较高的案例

# 3.1 中文姓名脱敏

该程序的运行结果如下：

历**

# 3.2 身份证号脱敏

该程序的运行结果如下：

51343***********1X

# 3.3 手机号脱敏

该程序的运行结果如下：

180****1999

# 3.4 邮箱脱敏

该程序的运行结果如下：

1*********@qq.com

# 3.5 银行卡号脱敏

该程序运行结果如下：

9559 **** **** *** 5810
