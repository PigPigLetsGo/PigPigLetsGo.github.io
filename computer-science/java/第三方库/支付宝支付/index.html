当调用 QRCodeUtil 类中的 main 函数时就会打印出一个二维码的 url 供访问，访问后就会跳转到支付宝的支付 controller 层接口中进行操作，它会获取 APPID，公钥，私钥 。然后向用户返回支付的信息。
