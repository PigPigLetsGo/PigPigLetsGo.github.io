# 图片文字识别

认识 OCR 什么是 OCR？

OCR （Optical Character Recognition，光学字符识别） 是指电子设备 (例如扫描仪或数码相机) 检查纸上打印的字符，通过检测暗，亮的模式确定其形状，然后用字符识别方法将形状翻译成计算机文字的过程

方案说明
百度 OCR收费
Tesseract-OCRGoogle 维护的开源 OCR 引擎，支持 Java，Python 等语言调用
Tess4J封装了 Tesseract-OCR ，支持 Java 调用
