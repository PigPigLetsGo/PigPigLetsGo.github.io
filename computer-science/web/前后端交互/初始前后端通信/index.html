# 前后端通信的过程

前后端的通信是在 ' 请求 - 响应 ' 中完成的

概念解释

前端：浏览器端

客户端：只要能和服务器通信的就叫客户端

后端：服务器端

通过终端向服务器发送请求： curl https//www.imooc.com

会响应回来一些数据但是不能被解析，终端能与服务器进行通信所以也是属于客户端的。

# 前后端通信方式

# 1. 使用浏览器访问网页

在浏览器地址栏输入网址，按下回车

# 2.HTML 的标签

浏览器在解析 HTML 标签的时候，遇到一些特殊的标签，会再次向服务器发送请求

比如：link ，img ，script ，iframe

还有一些标签，浏览器解析的时候，不会向服务器发送请求，但是用户可以使用它们向服务器发送请求

比如：a ，form
