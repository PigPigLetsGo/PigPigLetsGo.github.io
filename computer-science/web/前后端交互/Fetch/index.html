# Fetch

# Fetch 是什么

Fetch 也是前后端通信的一种方式。

Fetch 是 Ajax (XMLHttpRequest) 的一种替代方案，它是基于 Promise 的。

Ajax 的兼容性比 Fetch 好。

Fetch 中目前还没有 abort，timeout，这些想要使用还要我们去实现。
