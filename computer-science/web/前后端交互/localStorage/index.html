# localStorage 是什么

localStorage 也是一种浏览器存储数据的方式 (本地存储)，它只是存储在本地，不会发送到服务器端。

单个域名下的 localStorage 总大小有限制。

# localStorage 的存储期限

localStorage 是持久化的本地存储，除非手动清除 (比如通过 js 删除，或者清除浏览器缓存)，否则数据是永远不会过期的。

sessionStorage

当会话结束 (比如关闭浏览器) 的时候，sessionStorage 中的数据会被清空

# localStorage 键和值的类型
